# Urban Harvest Dashboard

A responsive admin dashboard UI made for the UI/UX Developer Fresher Assignment.

The project is built using React + Vite and focuses more on clean UI/UX, responsive layouts and reusable frontend components.

---

## Features

- Responsive dashboard layout
- Modern sidebar navigation
- Mobile hamburger menu
- Collapsible sidebar
- Login page with validations
- Demo login support
- Redux state management
- Product management page
- Add product modal
- Product search/filter
- Delete products
- Toast notifications
- Reusable components
- Smooth hover animations
- Fully frontend based (mock data)

---

## Tech Stack

- React JS
- Vite
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- Framer Motion
- React Hot Toast
- Lucide React Icons

## Setup Instructions

## Setup Instructions

Clone the repository

```bash
git clone https://github.com/officialmohitagarwal/urbanharvest
```

Go inside the project folder

```bash
cd urbanharvest
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

For production build

```bash
npm run build
```

To preview production build locally

```bash
npm run preview
```
